# Farcaster ENS

A lightweight Ethereum EIP-712 identity proof project built with ethers.js.

## Features

- EIP-712 Typed Data Signing
- Signature Verification
- Identity Proof Export
- Node.js + Termux Compatible

## Install

npm install

## Run

./run.sh

or

node identity.js
